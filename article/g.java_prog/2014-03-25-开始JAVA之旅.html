不得不去搞JAVA了，看来是

Post @ 2014-03-25

  • 朋友帮介绍的工作，不得不开始去搞JAVA了 :-(.

    •   网上投了几份简历，都石沉大海，可能是第一份工作并不是本行吧，所以就比较难找，辞职马上就到期了，下一份工作还是没有着落，朋友又要离开他现在的这家公司去单干，叫我想去的话要早点。没办法啊，我个人总觉得很是惭愧，因为我没学过JAVA，他们公司是做平板电脑的，用的是Android系统，早在Android刚出来那会，我记得是大二还是大三，自己折腾了好久，用NDK写了个C 版本的Hello World就没弄了,因为我还是觉得C很酷。现在朋友要走了，介绍我过去做技术，说真的，因为个人觉得技术没到一个高度或者是入门级的人吧，觉得很惭愧。但是朋友的一番好心不能辜负，姑且去试一试吧。所以翻开了压低的书籍，找到了那本JAVA教程，开始学了，有C和Python的基础，看起来不是太累。鼓励自己好好学吧，即便是去不成朋友的公司，也要做自己喜欢的了。我是一个不怎么爱去外面跑的人，现在找工作都很不愿意去跑，总觉得有点失败....最后还是要对我关心和照顾并介绍工作的朋友说声：谢谢。等过两天去看看吧，不适合再慢慢找了，边找边学JAVA....在Blog里已经增加了JAVA目录，记录下自己的学习历程吧，今天把代码写在的Main函数后面，一直提示Syntax token error，开始我还一直怀疑是类的继承和实例化有问题，一个字符一个字符去核对书本的代码，最后搞得想死，无意中发现了 '}' 放在了函数的前面，顿时恍然大悟.我的起点比别人低，但我不觉得那有什么问题.加油，努力! :-).....说实话，现在确实挺后悔当时没学JAVA的....现在要用到了，才不得不去学，，不过好一点的是，自己的计算机基础还可以，有了C和Python的经历，我相信不会太苦B......
